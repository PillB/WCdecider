PROCESSING STEPS (exact order, executable by student using only CSV + .py)
1. Load wc_june17_21_model_dataset.csv (20 rows aligned 1:1 with wc_2026_matches_june_17-21.csv).
2. For each row parse finetune_applied (string) -> apply_finetunes() returns numeric (opener_draw_boost=0.055 if Rule 21, minnow=1.16, caf_shrink=-60 if Rule19, finisher+30, rotation etc.).
3. Compute base: two_way_win_prob(Ea+Ha+Fa, Eb, ...) per AGENT D formula.
4. three_way_1x2 with closeness draw + opener boost.
5. expected_lambdas with minnow_resilience + rotation (tanh + k=0.0038, renormalize).
6. compute_ou_bt_ts (independent Poisson), compute_margin_prob (for -1), custom p_plus1 (for +1 Asian).
7. DC joint for combos per Rule 17 (p_not_loss * p_over * 0.92).
8. Rule 14 uplift (+2pp on longshots p<0.25) if flagged.
9. Look up screenshot o in prior_odds (only real from PNGs).
10. EV = (p_sel * o - 1) * 100. Classify strength per thresholds + sensitivities (aggr/base/cons run in full ensemble but raw here for transparency).
11. JSON + printed table for audit.
