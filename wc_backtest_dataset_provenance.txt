WCdecider Historical Development Dataset — Provenance and Limits
================================================================
File: wc_backtest_historical_dataset.csv
Repair script: scripts/repair_historical_world_cups.py
Current rows: 217
Status: development-only; not confirmatory and not profitability-eligible

CURRENT COMPOSITION
-------------------
- WC_2018_GROUP: 48 structurally validated group fixtures.
- WC_2022_GROUP: 48 structurally validated group fixtures.
- Supplementary international rows: 121.
- Obsolete embedded WC_2026_GROUP rows: removed. The production pipeline loads
  verified 2026 results from wc_2026_results_through_june21.csv instead.

WORLD CUP STRUCTURAL CONTROLS
-----------------------------
- Exactly 48 fixtures per tournament.
- Exactly 32 teams and three appearances per team.
- No duplicate unordered team pair.
- Frozen eight-group membership.
- Exactly the six unordered pairings within every four-team group.
- No cross-group pairing.
- Global chronological output ordering.

WORLD CUP RESULT CONTRACT STATUS
--------------------------------
The embedded 2018/2022 contracts were corrected after independent model-risk
validation found duplicate, missing, misdated, and incorrect fixtures. They are
internally complete and structurally validated. They remain interim because the
repository does not yet include a hash-bound authoritative raw FIFA result file
with row-level URL, retrieval timestamp, and locator for each fixture.

WORLD CUP ELO
-------------
Each tournament starts from the documented tournament-start snapshot values in
wc_backtest_historical_loader.py. Pre-match Elo is then replayed chronologically
with neutral-site K=20 updates after each prior fixture. Static tournament Elo
is no longer copied into every row.

SUPPLEMENTARY ROWS
------------------
The 121 friendlies/qualifier rows retain their prior source and walk-forward
construction. Their competition classification and source metadata remain
insufficient for confirmatory validation. They must be replaced or explicitly
regime-stratified before performance claims.

ODDS LIMIT
----------
Historical odds in this file are legacy proxies without verified quote
timestamps. They are excluded from primary profitability, ROI, CLV, or staking
validation.

REPRODUCTION
------------
Run:

  PYTHONDONTWRITEBYTECODE=1 python3 -B scripts/repair_historical_world_cups.py

The script preserves only the 121 supplementary rows, rebuilds both complete
World Cup group stages, sorts true parsed dates, and writes deterministic CSV.

REMAINING BLOCKERS
------------------
- Hash-bound authoritative fixture/result manifests.
- Row-level direct URLs and retrieval timestamps.
- Raw tournament-start Elo snapshots and hashes.
- Unified or regime-stratified Elo validation.
- Verified competition metadata for supplementary rows.
- A new prospective sealed holdout.

No current metric derived from this file is a confirmatory performance claim.
